hexo的基本操作

08 26 2020

请提前安装好Nodejs git,了解npm的用法以及markdown书写格式

安装

1、安装hexo

cnpm i hexo -g

2、进入空的项目文件进行初始化

hexo init

3、更换主题

在hexo官方网站上找到喜欢的主题,clone到themes目录下,并在站点配置文件(外层)_config.yml中更改主题配置
theme: rlflash rlflash是我选择的主题目录

使用

1、生成文件

hexo n "hexo基本操作"

2、本地调试

hexo s

3、生成静态文件

hexo g

配置

1、导航栏的生成

在主题配置文件(内层) _config.yml中更改配置

hexo n page about
